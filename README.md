# Full Stack Open

This repository contains my solutions and coursework for the [Full Stack Open](https://fullstackopen.com/en/) course offered by the University of Helsinki.

## About the Course
The course serves as an introduction to modern web application development with JavaScript. The main focus is on building single-page applications with ReactJS that use REST APIs built with Node.js. 

## Technologies Used
- HTML, CSS, JavaScript
- React
- Node.js & Express
- MongoDB
- Testing (Jest, Cypress)
- GraphQL
- TypeScript
